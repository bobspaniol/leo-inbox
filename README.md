# Inbox Leo

**URL:** https://bobspaniol.github.io/projeto-inova/inbox-leo/

## Como usar
1. Abre a URL no **Chrome** do celular
2. Toca em **Gravar** e fala o lançamento (ex: "mil reais pro Pedro, Gol branco")
3. Confere o texto transcrito — edita se precisar
4. Toca em **Parar gravação**
5. Toca em **Enviar pro Forms** → abre o Forms já preenchido → confere e envia

## Enviar o link pro Leo
Manda no WhatsApp: `https://bobspaniol.github.io/projeto-inova/inbox-leo/`
Leo pode salvar na tela inicial do Android: menu do Chrome → "Adicionar à tela inicial".

## Field IDs do Forms (pré-preenchimento)
- Tipo: `entry.836291884` → pré-selecionado como "Registro no consolidado"
- Conteúdo: `entry.954175182` → recebe o texto transcrito
